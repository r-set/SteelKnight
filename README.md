## Controls
| Key | Command |
| --- | --- |
| <kbd>A</kbd>, <kbd>D</kbd> or <kbd>←</kbd>, <kbd>→</kbd> | Move |
| <kbd>Z</kbd>, <kbd>↑</kbd>, <kbd>Space</kbd> | Jump |
| <kbd>X</kbd>, <kbd>K</kbd>| Attack|
| <kbd>Esc</kbd> | Menu|
